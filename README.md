## 📡 OVERVIEW

In a world drowning in digital chaos, **File Organizer v1.0** emerges as a silent guardian—an automated batch script engineered to restore order.

This lightweight utility scans your current directory and **reclassifies files based on type**, transporting them into designated storage sectors:

* 🖼️ Images
* 📄 PDFs
* 📝 Documents
* 🎬 Videos
* 🎵 Music
* 📦 Others

No UI. No noise. Just execution.

---

## ⚙️ HOW IT WORKS

Upon activation:

1. 🔧 **Creates categorized directories** (if they don't already exist)
2. 🚀 **Scans for known file extensions**
3. 🧲 **Moves files into respective folders**
4. 🧹 **Captures remaining unknown files into "Others"**
5. 🛡️ **Self-preservation protocol** prevents moving the script itself

---

## 🧪 SUPPORTED FILE TYPES

| Category  | Extensions                                |
| --------- | ----------------------------------------- |
| Images    | `.jpg`, `.jpeg`, `.png`, `.gif`           |
| PDFs      | `.pdf`                                    |
| Documents | `.doc`, `.docx`, `.txt`, `.xlsx`, `.pptx` |
| Videos    | `.mp4`, `.mkv`, `.avi`                    |
| Music     | `.mp3`, `.wav`                            |
| Others    | Everything else                           |

## 🚀 USAGE PROTOCOL

1. Place the `.bat` file inside the directory you want to organize
2. Double-click or execute via terminal
3. Watch the system restructure your files
4. Press any key to terminate

* 🧬 Designed for **local directory execution only**
